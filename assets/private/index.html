  • No sharing: URLs to private content should not be shared with unauthorized users

How This Works

This private directory is protected by Axum middleware that:

  1. Checks for a valid session in your request
  2. Verifies that you've completed WebAuthn authentication
  3. Allows access only if both conditions are met
  4. Logs all access attempts for security monitoring

The middleware implementation ensures that even if someone has a direct link to private content, they cannot access it without proper authentication.
