About This Demo

This WebAuthn demonstration includes:

  • Invite Code System: Registration requires valid invite codes
  • WebAuthn Authentication: Passwordless login using FIDO2
  • Public Static Files: Content accessible to everyone
  • Private Static Files: Content only for authenticated users

Note: To access private content, you'll need to register with an invite code and authenticate using WebAuthn.
